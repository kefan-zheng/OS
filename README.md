### 1.进程管理-电梯调度
有一栋共有20层的楼，有5部互相连接的电梯。每个电梯都有一些必要的功能键，如：数字楼层键、开门键、关门键、报警键。同时，在每个楼层，5部电梯拥有独立的数码显示屏显示电梯状态以及共享的上行键、下行键。同时，本项目还有如下假设：</br>
1.所有电梯初始状态都在第一层。</br>
2.每个电梯在没有相应任务的情况下停留在当前楼层。</br>
